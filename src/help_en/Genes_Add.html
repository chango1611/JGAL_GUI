Add Gene

Once the previous steps are made, you can add the gene to the chromosome, this can be done by pressing on the button that says "Add" (without the quotes). If the gene being added is valid, then it will be successfully added.

In some cases it may happen that a gene can not be added even though its configuration is valid. This happens when you have one of these genetic operations active:

  • Ordered Crossover

  • Swap Mutation

  • Shuffle Mutation

  • Inversion

And is because these operations require that all chromosome genes share the same configuration.

Some problems may need to create many genes that share the same configuration, for these cases we suggest using the Factory option.
