Itializers

There are two kinds of initializers:

  1. Classic

  2. Permutation

Classic

Consists of generating random values for the genes on chromosome without going through restrictions. It is the commonly used initializer.

Permutation

Consists of generating random permutations of the genes possible values. It has some restrictions for it to be used:

  • All genes must have the same configuration

  • All genes must be of Integer type

  • The maximum value minus the minimum value of each gene must be greater or equal than the size of the chromosome

Note. If this option is activated and a new gene is defined, this parameter will be automatically changed to the classic initializer if the new gene doesn't meet the conditions.
