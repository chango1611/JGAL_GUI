Arithmetic Operation

Arithmetic operations are all operations that can be performed between numerical objects. They are divided into simple operations (+, -, *, /,%, **) and complex operations

  • <Arithmetic Statement> ::= <Simple Arithmetic> | <Complex Arithmetic>

  • <Simple Arithmetic> ::= <Arithmetic Object> ((+ | - | * | / | % | **) <Arithmetic Object>)?

  • <Arithmetic Object> ::= (-)? (<Number> | <Identifier> | <Arithmetic Statement>)

  • <Number> ::= (0..9)+ | PI | E

  • <Complex Arithmetic> ::= <Operator>("("<Arithmetic Object> (,<Arithmetic Object>)*")")?

  • <Operator> ::= Abs | Max | Min | Root | Log | Sin | Cos | Tan | Asin | Acos | Atan | Hypot | toRadians | toDegrees | Round | Ceil | Floor | RandI | RandD | RandB | Bin2Int | Pos

Los <Arithmetic Object> can be Numbers, Identifiers or other arithmetic operation.

Complex Operations

The Complex operations, are the remaining operations that are not found in the simple operations, The number of parameters depends directly on the operation being performed.

The Complex Operations accepted are:

  • Abs(X): Returns absolute value of X

  • Max(X,Y): Returns X if X>Y, Y otherwise

  • Min(X,Y): Returns X if X

  • Root(X,Y): Returns the root base Y of X

  • Log(X,Y): Returns the logarithm base Y of X

  • Sin(X): Returns the sin of X

  • Cos(X): Returns the cosin of X

  • Tan(X): Returns the tangent of X

  • Asin(X): Returns the arco-sin of X

  • Acos(X): Returns the arco-cosin of X

  • Atan(X): Returns the arco-tangent of X

  • Hypot(X,Y): Returns the hypotenuse value which two other sides of the triangle are X and Y

  • toRadians(X): Returns the transformation of X from degrees to radians

  • toDegrees(X): Returns the transformation of X from radians to degrees

  • Round(X): Returns the closest integer to X

  • Ceil(X): Returns the smallest integer that is greater or equal than X

  • Floor(X): Returns the biggest integer that is less or equal than X

  • RandI(X,Y): Returns a random integer in the range [X..Y)

  • RandD(X,Y): Returns a random double in the range [X..Y)

  • RandB: Returns 0 o 1 randomly

  • Bin2Int(X1,X2...Xn): Returns the result of transforming a String of binary numbers represented by the concatenation of the bits Xi into an Integer

  • Pos($id): Returns the position of an identifier

e.g:

Max(5,3)+5;

The code above performs the Max between 5 and 3, returning the value 5 and then adds the value 5, returning 10.

Return

The arithmetic operations return the result of the operation.
