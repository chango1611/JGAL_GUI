Lineal Ranking

Is a type of selector by Ranking.

It is based on the lineal formula:
Prob(rank)= q - (rank-1)r

Where q must be within the range [1/pop_size,2/pop_size]

Requires the user to enter the value of q. Because the condition for q can not be validated for the moment, q value is found in the range [1,2], so that the real value would be q / tam_poblacion.
