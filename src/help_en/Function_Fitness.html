Fitness Function

The fitness function is used to calculate the fitness of the chromosomes of each generation, This function must return a double or GAL_Parser will throw an error.

Pressing on the "Define Fitness Function" button (without the quotes) opens a window like this one:

Each number is defined as follows:

  1. Code Area

  2. Calculator

  3. Complex Buttons

  4. Add Variable

  5. Validate

The fitness function can not be defined until the chromosome is defined.
Note. If changes are made on the chromosome, the code must be re-validated.
