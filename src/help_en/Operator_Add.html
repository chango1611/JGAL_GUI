Add Operator

Once the previous steps are made, you can add the operator, this is done by pressing on the button that says "Add" (without the quotes). If the operator being added is valid, then its successfully added.

In some cases it may happen that an Operator can not be added even though its configuration is valid. This occurs when you want to add some of these genetic operations:

  • Swap Mutation

  • Shuffle Mutation

  • Inversion

And is because these operations require that all chromosome genes share the same configuration.
