Add Many

The add many option lets you quickly create many genes that share the same configuration. Equivalent to using add multiple times.

This opens a window where you must enter how many copies you want to create. The names of each gene is the name that was placed followed by _Number.

Example: For the name "gene" and size of production 5, 5 genes are obtained:

gene_0, gene_1, gene_2, gene_3, gene_4.

Note. This option only adds, so it ignores if a gene is getting edited.
