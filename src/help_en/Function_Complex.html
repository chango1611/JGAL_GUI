Complex Buttons

This area displays buttons concerning complex instructions. Are subclassified into:

  1. Mathematics

  2. Trigonometry

  3. Conditionals/Cycles

  4. Logic

  5. Variables

  6. Others

1. Mathematics

It refers to mathematical operations that are not related to trigonometry. The operations are:

  • Root(X,Y): Returns the root base Y of X

  • X**Y: X powers Y

  • X%Y: X mod Y

  • Abs(X): Returns absolute value of X

  • Max(X,Y): Returns X if X>Y, Y otherwise

  • Min(X,Y): Returns X if X

  • Log(X,Y): Returns logarithm base Y of X

2. Trigonometry

It refers to math operations that are related to trigonometry. The operations are

  • Sin(X): Returns sin of X

  • Cos(X): Returns cosin of X

  • Tan(X): Returns tangent of X

  • Asin(X): Returns el arco-sin of X

  • Acos(X): Returns el arco-cosin of X

  • Atan(X): Returns la arco-tangent of X

  • Hypot(X,Y): Returns the hypotenuse value wich two other sides of the triangle are X and Y

  • toDegrees(X): Returns the transformation of X from radians to degrees

  • toRadians(X): Returns the transformation of X from degrees to radians

3. Conditionals/Cycles

It refers to the instructions for conditional and cycles. The operations are:

  • If, Then, Else, End: They are used to create conditionals

  • While, Do, End: They are used to create cycles

4. Logic

Contains all logic and comparison operations. The operations are:

  • True: The logic value True

  • False: The logic value False

  • Not: Denial of a boolean

  • &&: AND operation between two elements

  • ||: OR operation between two elements

  • ^: XOR operation between two elements

  • >: Greater than comparison

  • <: Less than comparison

  • =: Equal to comparison

Note. The operation: Greater of equal than(>=), Less of equal than(<=) and Different (<>); can be generated from the combination of logical buttons.

5. Variables

Contains the elements used to assign or use variables. It also includes symbols for creating Strings and Chars:

  • $: Used at the beginning of any identifier

  • {,}: Used to access an identifier by its position

  • := : Assignment Symbol

  • ": Used to create a String

  • ': Used to create a Char

  • Factory: Used to create a Char

Note. The logic behind the identifiers can be checked in the Sintaxis.

6. Others

It refers to operations that don't enter in previous classifications:

  • Round(X): Returns the closest integer to X

  • Ceil(X): Returns the smallest integer greater than or equal to X

  • Floor(X): Returns the bigest integer less than or equal to X

  • RandB: Returns 0 or 1 randomly

  • RandI(X,Y): Returns a random integer in the range [X..Y)

  • RandD(X,Y): Returns a random real in the range [X..Y)

  • RandC(X,Y): Returns a random character in the range [X..Y)

  • PI: Returns PI value

  • E: Returns E value
