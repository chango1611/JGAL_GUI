JGAL

JGAL is an application that allows the development of genetic algorithms, without the need for extensive knowledge in the field of programming.

This helps will guide the user through the steps required to develop a genetic algorithm successfully within the application, These steps include:

  • Define Chromosome

  • Define Function

  • Configure Algorithm

  • Run Algorithm

Framework

This program uses a framework with the same name "JGAL", if you wish to use the framework instead of the application, it can be downloaded from this address: https://github.com/chango1611/JGAL
