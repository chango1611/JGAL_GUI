Window Size

The window is used to store the last populations evaluated up to the moment, in order to use them on the definition of the termination condition.

The size of the window allows to define how many populations may be within the window.

It is important to know that the window size, changes the amount of constant values ??for the function.

e.g.

If you have a window size = 1, then it will have 3 constant values (popsize, bestFitness_0, averageFitness_0).
If you have a window size = 2, then it will have 5 constant values (popsize, bestFitness_0, averageFitness_0, bestFitness_1, averageFitness_1).

This means that if the window size is N, then there will be 2 * N +1 constants.

Note. bestFitness_0 represents the best fitness for the most recent population.
bestFitness_1 the second most recent... and successively.
