Assign

Assignments allow you to place a value to a variable. If the variable does not exist, create a new variable. You can not assign a value to constant value. Check Costant Values for more information.

  • <Assign Statement> ::= <Identifier> := (<Logical Statement> | <Arithmetic Statement> | <Identifier> | <String> | <Char>)

The <Identifier> lets you know to which variable will be done the assignment.

e.g:

$i:= 0+1;

The code above assigns the value 0+1 to the variable i.

Return

Assignments return the new value of the variable.
