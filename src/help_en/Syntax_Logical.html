Logical Operations

Logical operations are all operations that can be performed between logical objects such as AND ("&&") OR ("| |") and XOR ("^").

  • <Logical Statement> ::= <Logical Object> (("&&" | "||" | "^") <Logical Object>)?

  • <Logical Object> ::= (Not)? (True | False | <Comparison Statement> | <Identifier> | <Logical Statement>)

The <Logical Object> can be True, False, a Comparison, an identifier or other Logical Operation. Also they may be negated with a Not.

Comparisons

In GAL_Parser comparisons can be made between two objects by operators ("<" | ">" | "=" | "<=" | ">=" | "<>").

  • <Comparison Statement> ::= <Comparable Object> (("<" | ">" | "=" | "<=" | ">=" | "<>") <Comparable Object>)?

  • <Comparable Object> ::= <String> | <Char> | <Arithmetic Statement> | <Identifier>

This comparisons returnn True it they are fulfilled or False if not. Comparisons should be between elements of the same type or the parser will throw an error.

e.g:

(5>4) && True;

The code above performs the comparison 5>4 which returns True and then operates with AND against a True, returning True.

Return

The logical operations return the result of the operation.
