Run

By clicking the "Run" button (without the quotes), the user is asked which type of optimization would like to perform(maximization or minimization). After selecting an option it continues to run, if the implementation does not throw an error, then it displays the Results.

Note 1. The minimization requires that the fitness of the chromosomes be at most 1.000.000.000

Note 2. If something is missing to run, JGAL_GUI will throw a message saying that missing element in order to run
