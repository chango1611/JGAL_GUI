Array

Arrays can assign multiple variables in a faster way. They follow the syntax:

  • <Array Statement> ::= Array(<Identifier>,"{"{<Logical Statement> | <Arithmetic Statement> | <Identifier> | <String> | <Char>};"}")

The <Identifier> lets you know to which variable will be done the assignment.

e.g:

$Array($a,{5;True;3+3;});

The above code assigns the value 5 to the variable $a_0, True to the variable $a_1 and 6 to the variable $a_2.

Return

The arrays return the result of the last assignment.
