Cycles

Cycles are instructions that are performed repeatedly until the condition attached to the cycle no longer applies. For GAL_Parser cycles are handled with the <While Statement>.

  • <While Statement> ::= While <Logical Statement> Do <Function> End

The <Logical Statement> is the condition to be met before each repetition of the cycle, while the <Function> are the instructions to be evaluated each time the condition is met.

e.g.

$i:= 0;
While $i<10 Do
  $i:=$i+1;
End;

The code above starts by creating a variable i which is assigned 0. Then it enter a cycle where while i is less than 10 internal instructions are executed, which in this case would be an increase of the variable i by 1. Eventually i is equal to 10 in which case the cycle ends.

Return

Cycles return the last statement that was run internal to them. If the condition is false from the start, returns null.

Note. If the execution of a cycle is longer than 10 seconds, then it will throw a "Run Time Exceeded" error.
