Constants Values

Constants are identifiers that are defined before executing the function. These identifiers vary between the fitness function and the termination function

Constants of the Fitness Function

The constants for the fitness function are the genes of the chromosome thats being evaluated, to access them you can use the name or position of the gene.

Constants of the Termination Function

The constants for the termination function is the current population size, the best fitness and average fitness from the populations of the window. To access them:
popsize is used for the population size
bestFitness_i and averageFitness_i for the best fitness and average fitness (With i a number in the range of the window).

Note. i=0 represents the most recent population.
i=1 represents the second most recent population.... and successively.
