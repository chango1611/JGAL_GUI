Classic Handler

It's the commonly used handler when developing a Genetic Algorithm.

Follows the next program structure:

procedure Genetic Algorithm
  begin
   t = 0;
   initialize P(t);
   evaluate P(t);
   while (not condición de terminación) do
    t= t+1;
    select P(t) from P(t –- 1);
    alter P(t);
    evaluate P(t);
   end;
   return;
  end;

For the classic handler it must be define the population size and maximum number of generations
