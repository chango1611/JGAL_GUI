Conditional

Conditionals are instructions that can be executed or not depending on the value of a condition. For GAL_Parser conditionals are handled by the <If Statement>.

  • <If Statement> ::= If <Logical Statement> Then <Function> (Else <Function>)? End

The <Logical Statement> is the condition to be evaluated.

If the condition is true then it runs <Function> that is immediately after the Then.

If the condition is false, there are two options:

  • If the Else segment exists then runs <Function> that is immediately after the Else.
  • If the Else segment does not exist, then it skips the conditional.

e.g.

$i:= 20;
If $i<=10 Then
  $i:=10;
Else   $i:=0;
End;

The code above starts by creating a variable i which is assigned 20. Then it enters a conditional where if i is less or equal than 10 runs the Then segment, otherwise the Else segment. As the condition is false, the Else segment is executed and i is assigned 0.

Return

The conditionals return the last statement that was executed inside them. If the condition is false and there is no segment Else, returns null.
