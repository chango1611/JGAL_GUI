Definir Nombre del Gen

Cada gen puede tener el nombre que se desee, siempre y cuando cumpla con las siguiente condiciones:

  • No puede tener el mismo nombre que un gen que se definió anteriormente

  • Debe seguir el formato de un identificador. Es decir, inicia con un caracter alfabético o el caracter de subrayado, seguido de 0 o más caracteres alfa-numéricos o el caracter de subrayado
