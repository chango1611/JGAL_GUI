Asignación

Las asignaciones permiten colocar un valor a una variable. Si la variable no existe, se crea una nueva variable. No se le puede asignar un valor a los valores constantes. Ver Valores Constantes para más información.

  • <Assign Statement> ::= <Identifier> := (<Logical Statement> | <Arithmetic Statement> | <Identifier> | <String> | <Char>) |

El <Identifier> permite saber a que variable se le realizará la asignación.

Ejemplo

$i:= 0+1;

El codigo anterior le asigna el valor 0+1 a la variable i.

Retorno

Las asignaciones retornan el nuevo valor de la variable.
