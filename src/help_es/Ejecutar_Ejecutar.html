Ejecutar

Al presionar sobre el botón "Ejecutar" (Sin las comillas), se le pregunta al usuario que tipo de optimización desea realizar (Maximización o Minimización). Luego de seleccionar una opción se prosigue a ejecutar, si la ejecución no arroja algún error, entonces se muestra la pantalla de Resultados.

Nota 1. La minimización exige que la aptitud de los cromosomas sea como máximo 1.000.000.000

Nota 2. Si falta algún elemento para poder ejecutar, JGAL_GUI arrojará un mensaje diciendo que elemento falta para poder ejecutar
