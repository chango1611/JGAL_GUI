Agregar Varios

La opción agregar varios permite crear rápidamente muchos genes que comparten la misma configuración. Equivale a utilizar agregar múltiples veces.

Al pisar sobre el botón con el mismo nombre, se abrirá una ventana donde se debe ingresar cuantas copias se desea crear. Los nombres de cada gen será el nombre que se le coloco seguido de _Número.

Ejemplo: Para el nombre "gen" y cantidad 5, se obtienen los 5 genes:

gen_0, gen_1, gen_2, gen_3, gen_4.

Nota. Esta opción solo agrega, por lo que ignora si se está editando algún gen.
