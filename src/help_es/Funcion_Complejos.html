Botones Complejos

En esta área se muestran los botones referentes a instrucciones complejas. Se subclasifican en:

  1. Matemáticas

  2. Trigonometría

  3. Condicional/Ciclos

  4. Lógica

  5. Variables

  6. Otros

1. Matemáticas

Se refiere a operaciones matemáticas que no se relacionen a trigonometría. Las operaciones incluídas son:

  • Root(X,Y): Retorna la raíz base Y de X

  • X**Y: X elevado a la potencia de Y

  • X%Y: X mod Y

  • Abs(X): Retorna el valor absoluto de X

  • Max(X,Y): Retorna X si X>Y y Y en caso contrario

  • Min(X,Y): Retorna X si X

  • Log(X,Y): Retorna el logaritmo base Y de X

  • Bin2Int(X1,X2...Xn): Retorna el resultado de transformar una cadena de números binarios representado por la concatenación de los bits Xi a entero

2. Trigonometría

Se refiere a operaciones matemáticas que se relacionen a trigonometría. Las operaciones incluídas son:

  • Sin(X): Retorna el seno de X

  • Cos(X): Retorna el coseno de X

  • Tan(X): Retorna la tangente de X

  • Asin(X): Retorna el arco-seno de X

  • Acos(X): Retorna el arco-coseno de X

  • Atan(X): Retorna la arco-tangente de X

  • Hypot(X,Y): Retorna el valor de la hipotenusa cuyos otros dos lados del triángulo son X y Y

  • toDegrees(X): Retorna la transformación de X de radianes a grados

  • toRadians(X): Retorna la transformación de X de grados a radianes

3. Condicionales/Ciclos

Se refiere a las instrucciones para condicionales y ciclos. Las operaciones incluídas son:

  • If, Then, Else, End: Son usados para realizar condicionales

  • While, Do, End: Son usados para realizar ciclos

4. Lógica

Contiene todas las operaciones lógicas y de comparación. Las operaciones incluídas son:

  • True: El valor lógico True

  • False: El valor lógico False

  • Not: Negación de un valor lógico

  • &&: Operación AND entre dos elementos

  • ||: Operación OR entre dos elementos

  • ^: Operación XOR entre dos elementos

  • >: Operación mayor que

  • <: Operación menor que

  • =: Operación igual a

Nota. Las operaciones: Mayor Igual(>=), Menor Igual(<=) y Diferente (<>); pueden ser generadas a partir de la combinación de botones lógicos

5. Variables

Contiene los elementos usados para asignar o utilizar variables. Además, incluye los símbolos para la creación de Strings y Chars:

  • $: Utilizado al inicio de cualquier identificador

  • {,}: Utilizadas para acceder a un identificador por su posición

  • := : Símbolo de asignación

  • ": Utilizado para crear un String

  • ': Utilizado para crear un Char

  • Array($id,{valor1;valor2;..valor3;}): Usado para asignar multiples variables

  • Pos($id): Retorna la posición de un identificador pasado por parámetro

Nota. La lógica detras de los identificadores se puede ver en la Sintaxis.

6. Otros

Se refiere a aquellas operaciones que no entran en las clasificaciones anteriores:

  • Round(X): Retorna el entero más cercano a X

  • Ceil(X): Retorna el menor entero que sea mayor o igual a X

  • Floor(X): Retorna el mayor entero que sea menor o igual a X

  • RandB: Retorna 0 o 1 de manera aleatoria

  • RandI(X,Y): Retorna un entero aleatorio en el rango [X..Y)

  • RandD(X,Y): Retorna un real aleatorio en el rango [X..Y)

  • RandC(X,Y): Retorna un caracter aleatorio en el rango [X..Y)

  • PI: Retorna el valor PI

  • E: Retorna el valor E
