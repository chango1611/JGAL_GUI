Función de Terminación

La función de terminación es utilizada para calcular la condición de terminación por cada generación, esta función debe retornar un boolean o el GAL_Parser arrojará un error.

Al pisar sobre el botón "Definir Función de Terminación" (Sin las comillas) se abre una ventana como la siguiente:

Cada número es definido a continuación como:

  1. Espacio de Código

  2. Calculadora

  3. Botones Complejos

  4. Agregar Variable

  5. Tamaño de la Ventana

  6. Validar

La función de terminación se puede dejar vacía, de esta manera se utilizará la terminación por defecto "False;"
