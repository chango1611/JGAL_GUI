JGAL

JGAL es una aplicación que permite desarrollar algoritmos genéticos, sin la necesidad de tener amplios conocimientos en el ambito de la programación.

Esta ayuda guiará al usuario a través de los pasos necesarios para desarrollar un algoritmo genético de manera exitosa dentro de la aplicación, estos pasos incluyen:

  • Definir el cromosoma

  • Definir Función

  • Configurar Algoritmo

  • Ejecutar Algoritmo

Biblioteca

Este programa hace uso de una biblioteca que lleva el mismo nombre "JGAL", si desea utilizar la biblioteca en vez de la aplicación, se puede descargar desde esta dirección: https://github.com/chango1611/JGAL
