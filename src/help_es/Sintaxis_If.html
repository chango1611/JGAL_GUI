Condicional

Los condicionales son instrucciones que se pueden ejecutar o no en función del valor de una condición. En el caso de GAL_Parser los condicionales se manejan con el <If Statement>.

  • <If Statement> ::= If <Logical Statement> Then <Function> (Else <Function>)? End

El <Logical Statement> es la condición que se evalua.

Si la condición es verdad entonces se ejecuta el <Function> que se encuentra inmediatamente después del Then.

Si la condición es falsa, existen dos opciones:

  • Si el segmento del Else existe, se ejecuta el <Function> que se encuentra inmediatamente después del Else.
  • Si el segmento del Else no existe, entonces se salta el condicional por completo.

Ejemplo

$i:= 20;
If $i<=10 Then
  $i:=10;
Else   $i:=0;
End;

El codigo anterior inicia con la creación de una variable i a la que se le asigna 20. Luego se ingresa a un condicional, donde si i es menor igual que 10 se ejecuta el Then, sino el Else. Como la condición es falsa, se ejecuta el segmento del Else, donde a i se le asigna 0.

Retorno

Los condicionales retornan la última instrucción que se ejecutó interno a ellos. Si la condición es falsa y no existe el segmento del Else, retorna null.
